# Pong Game
![banner image] (screenshot/screenshot.png)

A basic pong game using SVGs and Javascript.


## Setup

**Install:**

`> npm i`

**Run:**

`> npm start`

## Keys

**Game config**
* g: New game
* n: New ball
* b: New ball bigger
* space bar: pause

**Player 1:**
* a: up
* z: down

**Player 2:**
* ▲ : up
* ▼: down
